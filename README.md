# Claude Vibekode

En enkel nettside laget med hjelp av [Claude Code](https://claude.ai/code).

## Innhold

- `index.html` — hovedsiden

## Kom i gang

Åpne `index.html` i nettleseren din, eller server den lokalt:

```bash
python3 -m http.server 8000
```

Besøk deretter [http://localhost:8000](http://localhost:8000).

## Bidra

1. Fork repoet
2. Lag en ny branch (`git checkout -b min-endring`)
3. Commit endringene dine (`git commit -m "Legg til noe kult"`)
4. Push til branchen (`git push origin min-endring`)
5. Åpne en Pull Request
